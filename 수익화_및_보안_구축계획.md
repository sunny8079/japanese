# 일본어 여행 회화 서비스 수익화, 트래픽 분석 및 보안 구축 계획

이 문서는 서비스 상용화에 따른 트래픽 분석 방안, 요금 폭탄 방지 대책, 광고 및 제휴 마케팅 확장 단계를 정리한 공식 가이드라인입니다.

---

## 1. 트래픽 분석 및 모니터링 방안
웹서비스에 얼마나 많은 사용자가 들어오고, 어떤 카테고리(인사, 교통, 쇼핑 등)를 주로 이용하는지 파악하여 광고 효율을 극대화하기 위한 도구들입니다.

### A. Google Analytics 4 (GA4) 연동 (가장 추천)
* **목적**: 방문자 수, 체류 시간, 모바일/PC 유입 경로, 클릭 이벤트 추적
* **작동 방식**: 구글 애널리틱스에 가입한 뒤 발급받는 고유 측정 ID (예: `G-XXXXXXXXXX`)를 `index.html` 상단에 심어둡니다.
* **장점**: 완전히 무료이며, 향후 구글 애드센스 광고 계정과 연결하면 **"어떤 페이지에서 가장 많은 광고 수익이 났는지"** 상세히 분석할 수 있습니다.

### B. Vercel Web Analytics 활성화
* **목적**: 사이트의 트래픽 추이 및 성능 점수(속도) 분석
* **작동 방식**: Vercel 대시보드에서 클릭 한 번으로 활성화하며, 별도의 코드 수정 없이 즉시 작동합니다.
* **장점**: 대시보드가 직관적이어서 하루 접속자 수 및 페이지뷰를 한눈에 모니터링하기 매우 편리합니다. (월 2,500건 이벤트까지 평생 무료)

## 2. 요금 폭탄 방지 및 보안 대책
공식 API 연동 시 발생할 수 있는 클라우드 과금 우려 및 외부 공격을 차단하기 위한 설정 가이드라인입니다.

### A. 구글 클라우드(GCP) - 일일 호출 할당량 제한 (최우선 차단막)
구글 클라우드 콘솔의 `IAM 및 관리자` ➔ `할당량 및 시스템 한도` 메뉴에서 **Text-to-Speech API의 하루 최대 사용량**을 고정합니다.
* **추천 제한값**: 하루 **50,000 글자 (Character)** 이하로 설정
* **효과**: 악의적인 공격자가 매크로를 이용해 수백만 번의 재생 요청을 보내더라도, 하루 5만 자 소모 시점에 구글이 API 서비스를 자동으로 임시 중단시킵니다. 따라서 **어떤 경우에도 무료 범위(월 400만 자) 내에서 안전하게 통제**됩니다.

### B. 구글 클라우드 - 실시간 예산 경고 메일 설정
GCP 콘솔의 `결제(Billing)` ➔ `예산 및 알림` 메뉴에서 예상 요금에 따른 경고를 설정합니다.
* **설정 한도**: 목표 예산을 **1,000원**으로 설정
* **효과**: 예상 청구액이 500원(50%), 900원(90%), 1,000원(100%)에 도달할 때마다 회원님의 구글 계정 이메일로 실시간 알림이 발송됩니다.

### C. Vercel API 키 보안 은닉
* **설정 방식**: 구글에서 발급받은 API 키를 Vercel 웹 호스팅 설정창의 **Environment Variables(환경 변수)**에 `GOOGLE_API_KEY`라는 이름으로 입력합니다.
* **효과**: 이 키는 오직 Vercel 서버 내부에서만 작동하며 외부 웹 브라우저로는 절대 노출되지 않습니다. 해커들이 웹페이지 소스 코드를 해킹하더라도 API 키를 훔쳐갈 수 없습니다.

## 3. 광고 및 제휴 수익화 확장 로드맵

### 1단계: 개인 맞춤화 완료 (현재 단계)
* **목표**: 어머님이 편하게 사용하실 수 있게 UI 다듬기, 번역 리스트 수정을 완료합니다.
* **수익 모델**: 없음 (안정성 확보에 집중)

### 2단계: 광고 연동 및 트래픽 분석 (준비 단계)
* **목표**: 서비스 공개 배포 및 방문자 수 수집 시작
* **구현 사항**: Google Analytics(GA4) 연동, Google AdSense 광고 영역(상/하단 배너) 확보 및 연동
* **수익 모델**: 클릭 및 노출형 배너 광고 수익 (애드센스)

### 3단계: 제휴 마케팅 최적화 (성장 단계)
* **목표**: 일본 여행이라는 명확한 관심사를 가진 방문자들의 실구매 전환 유도
* **구현 사항**: 카테고리에 맞는 제휴 마케팅 상품 링크 카드 배치
  - 교통 카테고리 ➔ Klook 일본 지하철/신칸센 패스 링크
  - 숙박 카테고리 ➔ 아고다/부킹닷컴 특가 검색창 및 배너 링크
  - 쇼핑/돈 카테고리 ➔ 쿠팡 파트너스 일본 돼지코/포켓와이파이 구매 링크
* **수익 모델**: 제휴 상품 구매 건당 수수료 정산 (가장 큰 수익원)

### 4단계: 공식 API 전면 전환
* **목표**: 우회 방식 차단 리스크 없이 100% 영구적으로 서비스 자동화 작동 보장
* **구현 사항**: 서버리스 API(`api/tts.js`) 활성화 및 구글 공식 클라우드 TTS 탑재
* **비용**: 광고 및 제휴 수익금의 일부로 구글 클라우드 요금 충당 (수익이 지출보다 훨씬 큼)
